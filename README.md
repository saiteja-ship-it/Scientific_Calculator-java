# Scientific Calculator (Java)

## Overview
A console-based scientific calculator built in Java to perform both basic arithmetic and advanced trigonometric operations.

## Key Features
- **Arithmetic Operations**: Addition, subtraction, multiplication, division.
- **Trigonometric Functions**: Sine, cosine, tangent.
- **Type Safety**: Enum-based operation handling for clean and maintainable code.
- **Robust Error Handling**: Graceful management of invalid inputs and division by zero.
- **Interactive Design**: Continuous calculation loop until user exit.

## Technologies Used
- Core Java
- Object-Oriented Programming (OOP)
- Enums
- Exception Handling

## Skills Demonstrated
- User-centric programming with friendly error handling
- Strong understanding of OOP principles
- Input validation and resilience in console applications
